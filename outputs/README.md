# Outputs

This folder contains visual outputs from the Global Earthquakes Visualisation project.

## Files

### `static_magnitude_map.png`

Static map showing global earthquake distribution, with marker size and colour used to represent magnitude.

### `static_depth_map.png`

Static map showing earthquake depth as an additional variable.

### `interactive_map_preview.png`

Screenshot preview of the interactive Folium map. The full interactive map is generated inside the project notebook.

## Purpose

These images are used in the main README so that viewers can understand the project outputs without opening the notebook first.
