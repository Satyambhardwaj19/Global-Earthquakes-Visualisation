# Notebooks

This folder contains the main analysis notebook for the project.

## Notebook

### `earthquake_analysis.ipynb`

This notebook includes the full workflow for:

- retrieving earthquake data from the USGS API
- converting GeoJSON data into a pandas DataFrame
- preparing geospatial data with GeoPandas
- creating static earthquake maps
- creating an interactive Folium map
- summarising key spatial patterns

## Note

The notebook is written as a portfolio version of the project, with cleaner structure, explanations, and outputs for GitHub.
